Parameters

Thresholds

The minimum and maximum thresholds of the image, resulting in a binary image of the given range. Depending on the type of algorithm selected (see the 'type' tab), the maxium value may be unused.

Type

Type of algorithm use to calculate the threshold. Check out the documentation for further information on each type. Speical algorithms such as THRESH_OTSU and THRESH_TRIANGLE are not included here since they are rarely used.
